# CoinCompass

> Turn your CoinStats crypto portfolio into an interactive map of the entire world — and find out whether your stack makes you a **King** or leaves you on the street, country by country.

CoinCompass reads your portfolio value from the **CoinStats API**, then paints a real, interactive world map (every country) with the lifestyle that amount buys you locally — from `Broke` to `King`. Hover any country for a quick read, click to zoom in and see the full verdict: how many years you'd live comfortably there, plus fun comparisons (years of rent, Big Macs, cups of coffee…).

![CoinCompass](docs/preview.png)

## Quick start

```bash
npm install
npm run dev
```

Then open http://localhost:5173.

- **Have a CoinStats key?** Paste it in and hit **Reveal my world** — your real portfolio value drives the map.
- **No key?** Hit the big **Try the Demo** button (or a preset like `$1M`) and explore instantly.

Once you're in, use the amount control in the header (`What if you had…`) to try any number and watch the whole map recolor live.

## Get a CoinStats API key

Grab a free key from the CoinStats developer portal:

### → https://coinstats.app/api/

The app calls `GET /portfolio/value` and reads the `totalValue` field. CoinStats serves `Access-Control-Allow-Origin: *`, so the browser calls the API **directly** — there is no backend and your key never leaves your machine.

**Advanced options** (in the gate, under _Advanced_):

| Option | Default | Notes |
| --- | --- | --- |
| Share token | _none_ | Optional `shareToken` query param (needed for some accounts/endpoints). |
| Endpoint base URL | `https://api.coinstats.app/v1` | Switch to `https://openapiv1.coinstats.app` if you prefer the OpenAPI host. |
| Currency | `USD` | Sent as `?currency=`. The cost-of-living model is USD-based, so **USD is recommended** for accurate verdicts. |

## How the lifestyle verdict works

For each country we estimate a realistic **comfortable single-person monthly cost** (incl. rent). Your amount buys some number of *years* of that lifestyle:

```
years = amount / (monthlyComfortable × 12)
```

…which maps to a tier:

| Tier | Years of comfortable living |
| --- | --- |
| King | 200+ |
| Elite | 50 – 200 |
| Living Large | 15 – 50 |
| Comfortable | 5 – 15 |
| Getting By | 1 – 5 |
| Scraping By | 0.25 – 1 |
| Broke | under 3 months |

Because the cost differs everywhere, the **same** amount crowns you a King in one country and leaves you Broke in another — that contrast is the whole point. Thresholds live in [`src/data/tiers.ts`](src/data/tiers.ts) and are easy to tune.

## The cost-of-living data

`src/data/countries.json` covers **every country** (250+ ISO entries). It's generated by [`scripts/build-countries-data.mjs`](scripts/build-countries-data.mjs):

1. Hand-set, Numbeo-style anchors for ~70 well-known countries.
2. Real **World Bank** GNI-per-capita (PPP) data (free, no key) fitted via log-log regression to estimate every remaining country.
3. Income-tier / region-median fallbacks so **no country is ever blank**.

Regenerate any time:

```bash
npm run data
```

> These are realistic estimates for a "what-if" experience, not authoritative quotes.

## Tech

- **Vite + React + TypeScript** — static SPA, no backend.
- **react-simple-maps** + **world-atlas** (Natural Earth 1:50m TopoJSON) — the real, interactive world map.
- **Framer Motion** — transitions, staggered reveals, the spring result card.
- **lucide-react** (thin strokes) + **flag-icons** (SVG flags) — no emoji.
- Bespoke CSS design system (`src/styles/`) — OLED-deep glassmorphism, animated aurora, gold/emerald/violet accents.
